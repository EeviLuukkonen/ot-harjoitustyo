# Käyttö-ohje

Peli alkaa suoraan onnistuneen asennuksen jälkeen terminaalin komennolla

```
poetry run invoke start
```

Peliä pelataan arvailemalla sanassa olevia kirjaimia ja klikkailemalla niistä ruudulla.
